변수, 상수

변수 -> 변하는 수

a라는 변수를 사용하고 싶다.
변수를 '선언'한다.

let a;

let a = 5;
console.log(a) ---> 5

a ="안녕하세요";
/*a = 10;
console.log(a) ---> 10*/

상수 -> 변하지 않는 수

a라는 상수를 사용하고 싶다.
상수를 '선언'한다.


const a = 5;
console.log(a) ---> 5

a = 10; ---------error
console.log(a) ---> 10? (x)



호이스팅
	: 선언 시에 선언부를 강제로 최상단으로 올려서 사용

	var(변수 ~ ESS)
		호이스팅을 합니다. 중복 사용 가능

	ex)
	*let의 경우
	console.log(a); -------error(참조 오류)
	let a = 5;

	*var의 경우
	console.log(b); --------------- undefined
	var b = 5; 

	var b
	-----최상단
	console.log(b) ------- undefined
	b = 5;

*let도 호이스팅 합니다.
하지만 호이스팅 시 참조 에러를 일으켜서 호이스팅이 안되는 것처럼 만든 것

자료형(Data Type)
	자바스크립트는 자료형을 명시하지는 않는다.
	하지만 자바스크립트도 엄연히 자료형을 가지고 있다.
	해당 자료형을 콘솔로 찍어볼 수 있으며
	숫자형에 문자열을 넣으면 에러가 나거나 인식을 못하는 경우가 있기 때문에 자료형을 알고 있고
	해당 자료형을 콘솔에 찍어서 비교할 수 있어야함.

	1. 숫자형(number)
		정수와 실수를 따로 구분하지 않고 모든 수를 하나의 자료형으로 표현
	2. 문자열(String)
		문자와 문자열 단위를 구분하지 않고 문자인 경우 하나의 자료형으로 표현

	3. undefined
		값이 할당되지 않음(정의되지 않았다), 자료형이 정해져있지 않음
	4. null
		값이 비어있다(" ", NULL) 빈 공간, 변수를 초기화하기 위해 사용

	5. 객체형(Object)
		강아지 1마리 
		이름: '콩이'
		나이: 3살
		몸무게: 4kg
	
		"콩이는 강아지구요 나이는 3살이며 몸무게는 4kg입니다."

		let animal = "DOG";
		let animalName = "콩이";

		let dog = {
			name: '콩이', (key: 'value',)
			age: 3,
			weight: "4kg"
			}

		여러 값을 하나의 변수에 저장할 수 있도록 해주는 타입
		구조는 키와 값(key -value)으로 이루어져있음 
		또한, 자바스크립트에서는 객체를 명시하기 위해 { }를 사용
		
		*객체의 하위 선택자( . )
			console.log(dog) ---> {name:"콩이",....} or object
			console.log(dog.name) ---> "콩이"


		*서버와 데이터 통신[주고 받을 때]을 할 때 가장 많이 사용하는 데이터 유형(JSON)
			객체와 형태가 똑같음(key = value)
			{
				"userId": "ksy"
			}
			백엔드한테 받는 데이터의 형태나 줘야하는 데이터의 형태가 보통은 
			JSON(객체와 유사한 데이터) 주고 받게 된다.
			
	
		*변수를 문자열에 포함시키기 위해서 어떻게 해야할까?
		"콩이는 강아지구요 나이는 3살이며 몸무게는 4kg입니다."
		1. console.log(dog.name + "는 나이는" + dog.age +
		"살이며 몸무게는+"dog.weight+"입니다.");
		2. console.log(dog.name, "는 나이는", dog.age,
		"살이며 몸무게는, "dog.weight, "입니다.");
		3. console.log(`${dog.name}는 나이는 ${dog.age}살이며 몸무게는
			${dog.weight}입니다.`);
